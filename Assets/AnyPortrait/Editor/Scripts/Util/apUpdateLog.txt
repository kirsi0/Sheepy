(영어)
<size=16>AnyPortrait Update Log v1.1.4/v1.1.5</size>
(Build December 24, 2018)


<size=13>Improvements</size>

- <color=blue>Extra Option</color> added
    Added Extra option that can be controlled by "Morph", "Transform" Modifiers
    Added ability to change Rendering Order in real time
    Added ability to change Image of meshes in real time
    <color=red>(For more information of the Extra option, please visit our homepage)</color>

- If the characters use the same Shader and Texture, they are merged into one material to reduce the Draw Call.
- Added "Refresh Meshes" button to refresh mesh in Inspector UI.
- Added functions to control the material of the mesh.
- In the upper UI, "Object Information UI unnecessary for the current situation" does not appear.



<size=13>Fixed issues</size>

- Fixed an issue where Draw Call Batching does not work normally.
- Fixed an issue where the draw call became too large when using a script function to control the material in batch.
- Fixed an issue where the parent mesh group would appear in the list of dialogs when adding a sub-mesh group to the mesh group.
- Fixed an issue where the default depth of mesh does not change on Unity 2018 (v1.1.5)

<size=16>AnyPortrait 업데이트 로그 v1.1.4/v1.1.5</size>
(2018년 12월 24일 빌드)


<size=13>개선된 사항</size>

- <color=blue>Extra 설정</color> 추가
    Morph, Transform 모디파이어로 제어할 수 있는 Extra 기능 추가
    렌더링 순서를 실시간으로 변경 가능
    메시의 이미지를 실시간으로 변경 가능
    <color=red>(Extra 설정 사용 방법은 홈페이지를 확인하세요)</color>

- 서로 다른 캐릭터가 동일한 쉐이더, 텍스쳐를 사용할 경우 재질을 공유하여 드로우콜을 최적화
- Inspector UI에서 메시를 갱신하는 "Refresh Meshes" 버튼 추가
- 메시의 재질을 제어하는 함수 3종 추가
- 상단의 UI에서 불필요한 객체 정보 UI가 출력되지 않도록 변경



<size=13>수정된 문제점</size>

- 드로우콜 배치(Drawcall Batch)가 정상적으로 수행되지 않는 문제가 수정됨
- 캐릭터의 모든 재질을 일괄적으로 제어하는 함수를 호출할 때, 드로우콜이 지나치게 증가하는 문제가 수정됨
- 하위 메시 그룹을 추가하는 다이얼로그에서 부모 메시 그룹이 보여지는 문제가 수정됨
- Unity 2018에서 메시의 기본 Depth를 수정할 수 없는 문제가 수정됨 (v1.1.5)

<size=13>改善点</size>

- <color=blue>Extra設定</color>追加
    「Morph」、「Transform」モディファイヤで制御することができる「Extra設定」が追加されました。
    レンダリング順序をリアルタイムで変更することができます。
    メッシュの画像をリアルタイムで変更することができます。
    <color=red>（Extra設定の使用方法は、ホームページをご確認ください）</color>

- Inspector UIでメッシュをリフレッシュするための「Refresh Meshes」ボタンが追加されました。
- メッシュの 「Material」を制御するいくつかの関数が追加されました。 (関連ページ)
- 上部のUIで「現在の状況について、不必要なオブジェクトの情報UI」は表示されません。



<size=13>修正された問題</size>

- 「Draw Call Batch」が正常に実行されない問題が修正されました。
- キャラクターのすべての「Material」を一括して制御するスクリプト関数を使用する際、「Draw Call」が過度に増える問題が修正されました。
- サブメッシュグループを追加するときに、ダイアログのリストに「親メッシュグループ」が出力される問題が修正されました。
-  Unity 2018でデフォルトのメッシュ深度が変わらない問題を修正しました。 (v1.1.5)
